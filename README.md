# Canvas Magic

AI 驱动的设计生成系统 - 让小白用户用一句话做出精美设计。

## 项目简介

Canvas Magic 是一个长期学习项目，通过 AI 代理从高质量 HTML 示例中学习，提炼出精美的 Canvas/Three.js/Shader 效果。

## 快速开始

```bash
# 克隆仓库
git clone https://github.com/YOUR_USERNAME/canvas-magic.git

# 进入项目
cd canvas-magic

# 查看当前版本
ls v0.1/
```
