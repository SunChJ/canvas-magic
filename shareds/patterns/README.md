# 设计模式库

此文件夹存放提炼的设计模式，跨版本可复用。

## 模式分类

### Three.js 模式
- `noise-sphere.glsl` - 噪声球体着色器
- `wave-plane.glsl` - 波浪平面着色器
- `particle-system.js` - 粒子系统模板

### Canvas 2D 模式
- `particle-network.js` - 粒子网络
- `dot-grid.js` - 点阵网格
- `plasma.js` - 等离子效果

### CSS 模式
- `crt-effect.css` - CRT 效果
- `glitch-text.css` - 故障文字
- `glow.css` - 发光效果

## 模式格式

每个模式文件包含：
1. 模式名称和描述
2. 核心代码
3. 使用示例
4. 注意事项
