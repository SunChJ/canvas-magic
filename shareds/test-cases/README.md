# 测试用例

此文件夹存放跨版本可复用的测试用例。

## 测试用例格式

每个测试用例是一个 Markdown 文件，包含：

```markdown
# 测试用例 TC-001

## 输入
用户描述："一个发光的紫色球体"

## 期望输出
- 效果类型：Noise Sphere
- 主色调：紫色系
- 交互：鼠标跟踪
- 动画：旋转 + 噪声位移

## 验证标准
- [ ] 单文件 HTML
- [ ] 响应式设计
- [ ] 60fps 动画
- [ ] 鼠标交互正常
- [ ] 视觉效果符合描述
```
