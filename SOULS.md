# SOULS.md - AI 代理的主动学习能力框架

## 核心理念

Canvas Magic 的 AI 代理不仅仅是一个代码生成器，而是一个**主动学习者**。它能够：
1. **观察**：从高质量示例中学习设计模式
2. **实验**：尝试不同的技术组合
3. **反思**：分析成功和失败的原因
4. **进化**：根据经验改进生成策略

## 学习算法

### 1. 模式匹配算法

```python
def match_pattern(user_input):
    """
    根据用户输入匹配最佳模式
    """
    # 关键词提取
    keywords = extract_keywords(user_input)
    
    # 模式匹配
    patterns = load_patterns()
    scores = []
    
    for pattern in patterns:
        score = calculate_match_score(keywords, pattern.keywords)
        scores.append((pattern, score))
    
    # 选择最佳模式
    best_pattern = max(scores, key=lambda x: x[1])[0]
    
    return best_pattern
```
